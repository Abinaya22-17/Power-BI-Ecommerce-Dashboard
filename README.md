# 📊 E-Commerce Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **E-Commerce Sales Dashboard** built using Power BI to analyze sales performance, customer trends, and product insights across different regions.

The dashboard helps stakeholders quickly understand business performance and identify growth opportunities.



## 🎯 Business Objectives

* Monitor overall sales and revenue performance
* Identify top-selling products and customers
* Analyze sales trends over time
* Compare performance across regions/countries
* Understand product demand patterns



## 🛠️ Tools \& Technologies Used

* Power BI (Data Visualization)
* Excel / CSV (Data Source)
* DAX (Data Analysis Expressions)



## 📂 Dataset Description

The dataset includes:

* Invoice / Order ID
* Order Date
* Customer ID
* Country / Region
* Product Description
* Quantity
* Sales Value



## 📊 Dashboard Highlights

### 🔹 1. Sales Trend Analysis

* Daily sales trend visualization using heatmap
* Helps identify peak sales periods and seasonality

### 🔹 2. Top Products Analysis

* Displays top-performing products based on revenue
* Helps in inventory and marketing decisions

### 🔹 3. Top Customers

* Identifies high-value customers contributing to revenue
* Useful for customer segmentation

### 🔹 4. Regional Sales Distribution 🌍

* Map visualization showing global sales distribution
* Helps identify strong and weak markets

### 🔹 5. KPI Metrics

* Total Sales
* Revenue distribution
* Customer contribution



## 📈 Key Insights

* A small number of products generate the majority of revenue
* Certain customers contribute significantly to total sales
* Sales are concentrated in specific countries/regions
* Sales patterns show variation across different time periods
